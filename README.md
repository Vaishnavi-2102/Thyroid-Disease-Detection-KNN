# Thyroid Disease Detection using KNN

This project implements thyroid disease classification using the K-Nearest Neighbors (KNN) algorithm in R.

## Dataset
The dataset contains thyroid hormone measurements such as:
- Age
- TSH
- T3
- TT4
- FTI
- Target class

## Visualizations
- Bar Chart (Class Distribution)
- Histogram (TSH Distribution)
- Boxplot (Outlier Detection)
- Scatter Plot (T3 vs TT4)
- Correlation Heatmap
- K vs Accuracy Plot
- Performance Metrics Chart

## Model
- Algorithm: KNN
- Train-test split: 80/20
- Distance metric: Euclidean
- K value: 5
